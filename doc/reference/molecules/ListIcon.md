# ListIcon
ListIcons are used to display icons in a list.

```example
<ResourceProvider>
  <ListIcon
    color="white"
    backgroundColor="blue"
    name="serving-filled"
  />
</ResourceProvider>
```
## Usage
| Name        | Type           | Description  |
| ----------- |:--------------:| ------------:|
|name **(required)**|string|
|iconColor|custom|
|backgroundColor|custom|
