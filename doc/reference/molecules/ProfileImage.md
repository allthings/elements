# ProfileImage
The ProfileImage is always used to show a profile image to the user.
```example
<Card>
  <ProfileImage image="https://placeimg.com/100/100/people" />
  <ProfileImage />
</Card>
```
## Usage
| Name        | Type           | Description  |
| ----------- |:--------------:| ------------:|
|onClick|func|
|children|node|
|image|string|
|showBorder|bool|<br>Default: true
|size|enum|<br>Default: 'medium'
