# Card
Cards can be used to group related content

```example
<Card>
 <CardContent>
   <Text size="xxl" strong>
     Cards
   </Text>
   <Text>
     Cards are the basic elements to fit content in. They can may
     contain any kind of content.
   </Text>
 </CardContent>
</Card>
```
## Usage
| Name        | Type           | Description  |
| ----------- |:--------------:| ------------:|
|children|node|
|containerStyle|object|
